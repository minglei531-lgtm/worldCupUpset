# 世界杯冷门生成器

一个纯前端的 Vue 3 + TypeScript + Vite 项目。用户选择一支强队和一支弱队，调整影响比赛结果的变量，系统通过透明规则模型和 10000 场蒙特卡洛模拟展示强队胜、平局、弱队冷门胜的概率，以及常见比分、因素贡献和自然语言冷门路径。

> 本项目为机制演示，不代表真实预测。参赛队名称采用 2026 世界杯阵容；能力值为产品演示参数，不是 FIFA 官方评级，不应用于投注或真实赛事判断。

## 技术栈

- Vue 3
- TypeScript
- Vite
- Composition API 与 `<script setup lang="ts">`
- Vitest
- 本地静态球队数据，无后端、无数据库、无外部 AI 接口

## 核心功能

- 从 48 支 2026 世界杯参赛队中选择强队与弱队
- 按 A–L 组真实小组赛赛程快速选择对阵，时间以北京时间显示
- 通过滑块调整：
  - 战术克制程度
  - 弱队防守执行力
  - 定位球优势
  - 门将超常发挥
  - 强队体能压力
  - 强队轻敌程度
  - 红牌或伤病扰动
- 点击“模拟 10000 场”后展示：
  - 强队获胜概率
  - 平局概率
  - 弱队冷门获胜概率
  - 高频比分分布
  - 冷门因素贡献
  - 冷门路径解释
- URL 自动保存球队、全部参数和模拟种子；分享链接可恢复同一结果
- 本地生成 1200 × 630 PNG 结果卡片，不依赖后端或第三方截图服务

## 模型原理

模拟引擎位于 `src/engine/`。

1. 根据强弱双方的实力、进攻、防守和稳定性生成基础预期进球。
2. 用比赛因素修正双方 xG：
   - 战术克制、防守执行和门将发挥主要压低强队 xG。
   - 定位球优势、强队轻敌、体能压力和扰动风险主要提升弱队 xG。
   - 弱队风格标签会给低位防守、反击和定位球路径提供小幅加成。
3. 每场比赛使用泊松分布抽样双方进球数。
4. 运行 10000 次后统计胜平负概率和比分分布。
5. 因素贡献用可解释的规则权重输出，正值代表更利于冷门，负值代表更不利于冷门。

模型有明确边界约束，避免极端参数导致明显失真的 xG 或比分结果。它追求逻辑自洽和交互解释性，而不是现实预测精度。

## 数据来源

- 48 支参赛队与 72 场小组赛赛程采用 ESPN 2026 FIFA World Cup 数据快照（2026-06-17）。
- 球队能力值、风格标签和模型权重由本项目为机制演示设定，不是官方数据。
